# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-15

### Added

- Initial release of Chopper's Hanoi Meal Timetable
- Interactive monthly calendar view for tracking meals throughout the year
- Random meal selector from 52 authentic Hanoi Vietnamese dishes
- Accept/Reject workflow for meal selection
- Beautiful meal images displayed in modal and calendar cells
- Today's date highlighting with distinctive pink border
- Responsive design supporting desktop, tablet, and mobile devices
- Smooth animations and micro-interactions
- WCAG-compliant accessibility features
- Tony Tony Chopper character animation in hero section
- Month navigation with Previous/Next buttons
- Meal information display with descriptions
- Hover effects and visual feedback for interactive elements

### Features

- **Calendar Interface**: 12-month view with 7-day week layout
- **Meal Database**: 52 curated Hanoi Vietnamese meals with images and descriptions
- **Modal Selector**: Beautiful modal popup for meal selection
- **Visual Feedback**: Clear visual states for empty, filled, and today's date cells
- **Responsive Layout**: Adapts to different screen sizes
- **Accessibility**: Keyboard navigation, screen reader support, proper semantic HTML

### Technical

- Built with React 19 and TypeScript
- Styled with Tailwind CSS 4
- UI components from shadcn/ui and Radix UI
- Client-side routing with Wouter
- Vite for fast development and building
- pnpm for package management

### Design

- Warm Playfulness with Medical Precision design philosophy
- Color palette inspired by Tony Tony Chopper
- Smooth animations and transitions
- Clear visual hierarchy and organization
